# Reporte final ejecutivo

Archivo principal:

```bash
final_executive_report.tex
```

Compilación recomendada desde esta carpeta:

```bash
latexmk -pdf final_executive_report.tex
```

Alternativa:

```bash
pdflatex final_executive_report.tex
pdflatex final_executive_report.tex
```

Las figuras PDF están en:

```bash
../../outputs/final_report/figures/
```

El script que genera las figuras está en:

```bash
../../scripts/final_report/make_final_report_figures.py
```
