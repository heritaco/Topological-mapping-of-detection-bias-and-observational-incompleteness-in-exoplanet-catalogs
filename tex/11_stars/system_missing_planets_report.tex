\section*{Conclusi\'on}
Este subproyecto no afirma descubrimientos de planetas reales ni detecciones definitivas. Su aporte es convertir una se\~nal topol\'ogica global de posible incompletitud observacional en un ranking intra-sistema de intervalos orbitales prioritarios para seguimiento, combinando arquitectura orbital, soporte topol\'ogico previo, an\'alogos globales y dificultad relativa de detectabilidad.

Los tres casos finales muestran que el m\'etodo puede pasar de una se\~nal topol\'ogica global a hip\'otesis observacionales concretas: un sistema, un intervalo orbital, una posici\'on candidata y una raz\'on plausible de no detecci\'on.

\[
\boxed{
\text{Mapper global}
+
\text{sombra observacional}
+
\text{arquitectura intra-sistema}
\Rightarrow
\text{ranking de intervalos orbitales prioritarios}
}
\]
